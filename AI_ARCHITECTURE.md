# «Без Б» — архитектура AI-слоя, модели и смета

Документ описывает, как AI встраивается в уже построенный продукт (FastAPI + бот +
Mini App + мультипортфель на VPS), какие фичи через Claude напрямую, какие через
OpenClaw, какие модели и сколько это стоит по месяцам.

Цены Claude API (за 1 млн токенов): Haiku 4.5 (`claude-haiku-4-5`) $1/$5;
Sonnet 4.6 (`claude-sonnet-4-6`) $3/$15; Opus 4.8 (`claude-opus-4-8`) $5/$25.
Кэш промпта: чтение ~0.1×, запись 1.25×; Batches −50%.

---

## 1. Где Claude напрямую, где OpenClaw

- **Claude напрямую из нашего FastAPI** — для продуктовых фич, привязанных к нашим
  данным портфеля: один запрос → структурированный ответ. Полный контроль промпта,
  формата и стоимости. Сюда идут: разбор сделки, разбор портфеля, сценарии.
- **OpenClaw (на VPS)** — для разговорного ассистента подписчикам и автономного
  ресёрч-агента (многошаговое: собрать данные → проанализировать → черновик). Это
  оркестратор; «мозг» внутри — тот же Claude (настраиваем OpenClaw на Claude).
  Вводим во 2-й фазе (нужно ~4 ГБ RAM → апгрейд VPS).

Принцип: **Фаза 1 — всё через прямой вызов Claude** (просто, дёшево, управляемо).
OpenClaw подключаем, когда появится спрос на живой чат-ассистент.

## 3. Рычаги удержания стоимости
1. **Haiku 4.5** на всё частое и короткое (быстрый разбор, чат) — в 3–5× дешевле.
2. **Кэш промпта**: системный промпт + контекст портфеля кэшируются → чтение ~0.1×.
3. **Batches API −50%** для не-срочного (ночные дайджесты).
4. **Sonnet/Opus** только на глубокий премиум-анализ.
5. Разбор сделки — низкий объём (DCA раз в 2 недели) → копейки.

## 4. Смета по аудитории (Claude API, в месяц)

Допущения на пользователя/мес: free — ~2 быстрых разбора (Haiku) + ~5 сообщений
чата; premium — ~10 глубоких разборов (Sonnet) + ~5 сценариев + ~20 сообщений.
Контент владельца (сделки + дайджест) ≈ **$2–3/мес** независимо от аудитории.

| Сценарий | Юзеров | Премиум | Free-часть | Premium-часть | Контент | **Итого Claude/мес** |
|---|---|---|---|---|---|---|
| **Старт** | 100 | ~10 | ~$2.7 | ~$4.8 | ~$2.5 | **~$10** |
| **Рост** | 1 000 | ~80 | ~$28 | ~$38 | ~$3 | **~$70** (с кэшем/батч ~$50) |
| **Масштаб** | 5 000 | ~400 | ~$138 | ~$190 | ~$5 | **~$330** (оптимизировано ~$200) |

Сопоставление с выручкой на масштабе: 400 премиум × 990 ₽ ≈ **396 000 ₽/мес**
(~$4 000). AI-затраты ≈ **5–8% выручки** — здоровая экономика. На старте $10/мес
вообще незаметно.

### Прочие расходы (из прошлого разбора)
- **VPS Beget**: ~700 ₽/мес (2 ГБ); при OpenClaw-агенте → ~1400 ₽/мес (4 ГБ).
- **Новости/макро**: $0 через web_search Claude или бесплатные тарифы; платный API
  по желанию.
- **Платёжный шлюз USDT** (CryptoCloud): ~0.4–1% с платежа, без абонплаты.
- **Домен**: опц. ~$10–15/год.

## 6. Порядок внедрения
1. **Фаза 1 (без новостей, без OpenClaw):** B разбор портфеля + A разбор сделки —

2. **Фаза 2:** дайджест + сценарии + переводчик рынка (добавляем web_search/новости).
