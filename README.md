# Без Б — публичный портфель (Telegram-бот)

MVP бота, который ведёт реальный инвестпортфель и показывает котировки,
состав и рост в реальном времени — в двух валютах (USD и RUB) с разбивкой
прибыли на «рост активов» и «курсовую разницу».

## Что умеет
- `/portfolio` (`/p`) — сводка: стоимость и прибыль в $ и ₽, разбивка по позициям
- `/chart` — круговая диаграмма состава + график роста
- `/price AAPL` — текущая цена актива и курс ЦБ
- `/buy AAPL 5 180` — записать покупку (только владелец)
- `/sell AAPL 2 200` — записать продажу (только владелец)
- `/snapshot` — зафиксировать точку стоимости (для графика роста)
- `/publish` — выложить сводку и графики в канал
- автопубликация в канал каждый день в 18:00 (если задан `CHANNEL_ID`)

Источники данных: акции/ETF/крипта — Yahoo Finance (yfinance), курс USD/RUB —
официальный API ЦБ РФ. Всё бесплатно, без API-ключей.

## Установка (Windows, PowerShell)

```powershell
cd "C:\Users\belog\Desktop\Новая тема\bezb_bot"

# 1) виртуальное окружение
python -m venv .venv
.\.venv\Scripts\Activate.ps1

# 2) зависимости
pip install -r requirements.txt

# 3) настройки
Copy-Item .env.example .env
# открой .env и впиши BOT_TOKEN и ADMIN_ID
```

### Где взять данные для .env
1. **BOT_TOKEN** — напиши `@BotFather` в Telegram → `/newbot` → получишь токен.
2. **ADMIN_ID** — напиши `@userinfobot` → пришлёт твой числовой id.
3. **CHANNEL_ID** — создай канал, добавь бота администратором. Укажи `@имя_канала`
   (для публичного) или числовой `-100...` id (для приватного). Можно позже.

## Запуск

```powershell
python bot.py
```

Бот стартует в режиме polling — работает, пока открыт терминал.
Для работы 24/7 позже перенесём на VPS (отдельная инструкция).

## Первые шаги после запуска
```
/buy AAPL 5 180      # купил 5 акций Apple по $180
/buy BTC 0.05 65000  # купил 0.05 BTC по $65000
/portfolio           # смотрим сводку
/snapshot            # фиксируем точку (делай раз в день для графика роста)
/chart               # диаграммы
```

## Дисклеймер
Бот показывает личный портфель автора и не является индивидуальной
инвестиционной рекомендацией.
